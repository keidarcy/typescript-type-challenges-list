# Get Return Type

## introduction

Implement the advanced util type `UnionToIntersection<U>`

For example

```ts
type I = Union2Intersection<'foo' | 42 | true>; // expected to be 'foo' & 42 & true
```

type UnionToIntersection<U> = (U extends any ? (arg: U) => any : never) extends (
  arg: infer I
) => void
  ? I
  : never;
