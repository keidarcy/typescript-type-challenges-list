# Get Return Type

## introduction

Implement the built-in `ReturnType<T>` generic without using it.

For example

```ts
const fn = (v: boolean) => {
  if (v) return 1;
  else return 2;
};

type a = MyReturnType<typeof fn>; // should be "1 | 2"
```

## start point

```ts

  bar: 'hello';
  prev(): number;
};

const fn = (v: boolean) => (v ? 1 : 2);
```
